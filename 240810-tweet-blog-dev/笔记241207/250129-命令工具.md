
## tree命令查看目录结构
使用 `tree` 命令可以在命令提示符（CMD）中查看目录结构。以下是详细说明：

### 基本命令
```sh
tree /F /A
```
- `/F`：显示每个文件夹中的文件。
- `/A`：使用 ASCII 字符而不是扩展字符来显示树结构，这在某些终端或文本文件中更易读。

### 其他命令
```
dir /S /B
```
- `/S`：递归列出所有**子目录及其中的文件**。
- `/B`：（Bare format）**仅显示路径，不显示额外信息**（如文件大小、创建时间等）。

要获取本目录下内容，只用 `/B` 即可
```
dir /B
```
