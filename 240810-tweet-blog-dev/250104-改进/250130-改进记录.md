### 解决编译包括问题
只有src中的ts文件才是需要编译的，其他的像 `drizzle.config.ts` 都是只会开发环境使用
```json
// tsconfig.json
{
  "compilerOptions": {
	// ...
    "outDir": "./dist",
    "rootDir": "./src",
  },
  "include": ["src/**/*"],
  "exclude": ["node_modules", "dist", "drizzle.config.ts"],
}
```

ESLint对于src之外的问题
```json
// .eslintrc.json
{
	// ...
    "ignorePatterns": ["drizzle.config.ts"]
}
```
