# 🦾 1. 伺服电机（Servo）是什么类型？

这里说的 Servo 指的是：

- **PWM 控制的舵机（SG90、MG996R、DS3218 等）**
- **机械臂常用的 180° 或 270° 舵机**
- **多轴机械臂（6DOF、7DOF）**

这些都是 **Arduino/ESP32 的强项**。

不是指：

- 工业伺服电机（AC Servo）
- 工业步进伺服（闭环）

工业伺服需要驱动器，不是 MCU 直接控制的范畴。

但机械臂玩具、桌面机械臂、教育机械臂 → 完全没问题。

# 🧩 2. Firmata 是否支持 Servo？

## ✔️ 支持，而且是官方内置功能

Firmata 协议本身就包含 Servo 控制指令。

Johnny‑Five 也内置 Servo 类：

```ts
import { Board, Servo } from "johnny-five";

const board = new Board();

board.on("ready", () => {
  const servo = new Servo(9);
  servo.to(90);
});
```

这就是完整代码。
